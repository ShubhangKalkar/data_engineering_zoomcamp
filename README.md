# Data Engineering Zoomcamp 2026

This repository contains my coursework, homework solutions, and hands-on projects from the **Data Engineering Zoomcamp 2026** by DataTalksClub.

The course covers end-to-end data engineering concepts including containerization, SQL, cloud infrastructure, orchestration, data warehousing, analytics engineering, batch processing, and streaming systems.

---

## Course Structure & Progress

| Week | Topic | Folder |
|-----:|-------|--------|
| Week 1 | Docker, SQL, Terraform | `week_01_docker_sql_terraform/` |
| Week 2 | Workflow Orchestration | `week_02_workflow_orchestration/` | 
| Week 3 | Data Warehouse & BigQuery | `week_03_data_warehouse/` | <-- In Progress
| Week 4 | Analytics Engineering (dbt) | `week_04_analytics_engineering/` |
| Week 5 | Batch Processing (Spark) | `week_05_batch_processing/` |
| Week 6 | Streaming (Kafka) | `week_06_streaming/` |

Each weekly folder contains:
- Homework answers
- SQL queries and scripts
- Docker / Terraform / pipeline configurations
- Clear instructions to reproduce results

## How to Use This Repository

3. Run the provided scripts and SQL locally
